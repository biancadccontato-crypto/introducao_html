Introdução

O que é a web?
A Web é um sistema de informação global que funciona sobre a infraestrutura da internet, operando através de um modelo onde navegadores (**browsers**) atuam como interfaces gráficas para solicitar e renderizar conteúdos interconectados. Sua origem está profundamente ligada à evolução da **ARPANET**, que estabeleceu os protocolos de comunicação de rede fundamentais, permitindo que o tráfego de dados — incluindo tecnologias precursoras como o **e-mail**, que introduziu a troca de mensagens entre servidores — evoluísse para o sistema de hipertexto que conhecemos hoje. Assim, a Web integra a conectividade herdada da ARPANET, a lógica de armazenamento e envio dos **servidores de e-mail** e a capacidade de interpretação dos **browsers** para transformar códigos complexos em uma experiência visual e acessível para o usuário final.

Funcionamento da Web
A Web funciona por meio de um modelo chamado cliente-servidor, onde o seu navegador (cliente) envia uma solicitação via protocolo HTTP ou HTTPS para um computador remoto (servidor) após localizar o seu endereço único através do sistema de DNS, que traduz nomes de domínios em endereços IP. Assim que o servidor recebe esse pedido, ele processa a requisição e devolve os arquivos necessários, como documentos HTML, folhas de estilo CSS e scripts JavaScript, que são então interpretados e renderizados pelo navegador para montar a interface visual e interativa que você acessa na tela.

O que é HTML?
Hypertext Markup Language, ou simplesmente HTML, é uma linguagem de marcação utilizada para representar informação na Web. Nasceu juntamente com o conceito da Web e hoje, conjuntamente às linguagens JavaScript e CSS, compõe as bases para o desenvolvimento de soluções frontend neste ecossistema. Desde seu desenvolvimento, em 1989, a linguagem HTML foi passando por diversas revisões, que culminaram com o lançamento de novas versões da linguagem. Atualmente está em uso a versão 5, motivo pelo qual é comum encontrar citações à linguagem como HTML5. Antes de propriamente iniciarmos nosso estudo na linguagem HTML5, precisamos entender os atores principais da Web e como eles se comunicam. Obviamente faremos uma simplificação da arquitetura, enfatizando o papel do web browser ou navegador e do web server. Esse processo de comunicação é feito através de um cliente e um servidor, o cliente envia uma requisição, uma solicitação de recurso, que é recebida pelo servidor e devolvida ao cliente por meio de uma resposta. Esse modelo de arquitetura distribuída é chamada cliente/servidor client/server e define em alto nível como ocorre a comunicação entre web browsers (clientes) e web servers (servidores).

Conceitos Básicos
O HTML (Hypertext Markup Language) é a linguagem de marcação fundamental da Web, funcionando como o esqueleto que define a estrutura e o significado de um documento através de uma hierarquia de **tags**. Cada elemento HTML instrui o navegador sobre como organizar o conteúdo, seja identificando títulos, parágrafos, imagens ou links, utilizando **atributos** para fornecer informações adicionais sobre esses componentes. Por ser uma linguagem semântica, o HTML moderno foca na organização lógica da informação, garantindo que tanto usuários quanto motores de busca compreendam a função de cada parte da página antes mesmo da aplicação de estilos visuais ou comportamentos interativos.
